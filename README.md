Генератор случайных паролей (Random Password Generator)

Автор: Галиев Раниль

ОПИСАНИЕ
Приложение с графическим интерфейсом для создания безопасных паролей.
Основные возможности:
- Выбор длины пароля от 4 до 32 символов.
- Выбор состава пароля: цифры, буквы, спецсимволы.
- Автоматическое сохранение истории генераций в файл JSON.

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
1. Тест на цифры: Выставить длину 6 и выбрать только "Цифры". 
   Результат: пароль типа 573921.
2. Тест на надежность: Выставить длину 16 и включить все чекбоксы. 
   Результат: сложный пароль типа f9#L2!pQ8*zR1&mN.
3. Проверка истории: Все сгенерированные пароли сразу отображаются 
   в таблице в нижней части окна.
