Генератор случайных паролей (Random Password Generator)

Автор: Галиев Раниль

ОПИСАНИЕ
Приложение для создания безопасных паролей с графическим интерфейсом.
Основные возможности:
- Настройка длины (4-32 символа) и типов знаков.
- Хранение истории последних генераций в формате JSON.
- Улучшенная обработка ошибок при работе с файлами.

ЗАВИСИМОСТИ
- Python версии 3.x.
- Стандартные библиотеки: tkinter, random, string, json, os.
- Требуется установленный пакет Tkinter (включен в стандартную поставку Python для Windows).

КАК ЗАПУСТИТЬ
1. Убедитесь, что Python установлен.
2. Скачайте файл main.py.
3. Откройте терминал или командную строку в папке с файлом.
4. Введите команду: python main.py

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ
1. Тест на длину: выставить 10, выбрать все типы символов.
   Результат: пароль типа "aB3!hN9@kL".
2. Тест истории: сгенерированный пароль сразу отображается в таблице снизу.
   Приложение логирует ошибки доступа к файлу истории в консоль.
